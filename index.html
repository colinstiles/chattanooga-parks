Welcome to Chattanooga Parks & Trails Explorer

What it shows: This interactive map displays public parks and trails throughout Chattanooga, Tennessee, with detailed information on amenities, trail types, lengths, and accessibility features.

How to use: Toggle between different base maps, click on parks (green circles) and trails (orange lines) for details, use the control panel to reset the view, and turn layers on/off using the layer control.

Data Source: Geometry data sourced from Tennessee TREC (Tennessee Research and Creative Exchange).

Disclaimer: This web map is for informational purposes only and is not intended to be used as an authoritative source for navigation or official planning.
